# Memory Dungeon - Product Context

## Why This Project Exists
Memory Dungeon combines the satisfying loop of memory matching with the progression fantasy of a dungeon crawler. It aims to make a simple card-matching mechanic feel epic through layered systems, atmospheric theming, and meaningful player choices.

## Problems It Solves
- Fills the gap between casual match-3 games and hardcore roguelikes — accessible but deep
- Provides replayability through daily challenges, artifacts, and New Game+
- Offers a complete mobile game experience in a single, focused scope

## How It Works
1. **Menu**: Player chooses to continue campaign, start new game, enter daily challenge, or visit the shop
2. **Dungeon Selection**: Choose from 6 unlocked chambers (Stone → Forest)
3. **Gameplay**: Cards are generated per level with increasing grid sizes (4×3 → 6×6)
   - Cards face-down during 1.2s preview, then flip back and unlock
   - Match pairs to earn score, coins, and progress toward clearing the level
   - Different card types trigger unique effects (heal, poison, gem shatter, scroll reveal, etc.)
4. **Progression**: Clear levels → earn coins → unlock artifacts → take on harder dungeons
5. **Daily Challenges**: Seeded deterministic puzzles reset daily with unique modifiers
6. **Deeper Descent**: After clearing all 6 dungeons, unlock New Game+ with larger grids and bonus rewards

## User Experience Goals
- **Satisfying feedback loop**: Sound effects, visual effects, streak counters
- **Clear difficulty curve**: Each dungeon ramps up with new card types and modifiers
- **Meaningful choices**: Artifact shop lets players customize their play style
- **Atmospheric immersion**: Themed visuals, ambient audio, dungeon descriptions
- **Polish**: Tutorial on first launch, tips overlay, colorblind mode, pause functionality
